# Sistema Inteligente de Diagnóstico de Riego (Gemini AI)

Este proyecto propone una solución de ingeniería que integra la adquisición de datos de sensores físicos (telemetría de variables hídricas) con la capacidad analítica de la Inteligencia Artificial de Gemini para actuar como un módulo de diagnóstico predictivo y mantenimiento industrial en tiempo real.

## Requisitos de Instalación
Para ejecutar este programa de forma local, clona este repositorio e instala la biblioteca oficial de Google:

```bash
pip install google-generativeai
